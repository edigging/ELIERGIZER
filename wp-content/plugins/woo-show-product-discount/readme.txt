== Description ==

**Woo Show Product Discount** an WooCommerce extension to show product discount on shop page as well as on product page. Most importantly provides lots of customization options to give it your unique touch.

= Features =

* **Show Product Discount On Shop Page**
Shows product discount on shop page. Setting to enable/disable this feature.
* **Show Product Discount On Product Page**
Shows product discount on product page. Setting to enable/disable this feature.
* **Show Product Discount On Sale Flash Section**
Shows product discount on sale flash section. Setting to enable/disable this feature.
* **Customizable Text To Show Product Discount On Shop Page**
Gives admin the freedom to customize the text for showing product discount on shop page.
* **Customizable Text To Show Product Discount On Product Page**
Gives admin the freedom to customize the text for showing product discount on product page.
* **Customizable Text To Show Product Discount On Sale Flash Section**
Gives admin the freedom to customize the text for showing product discount on sale flash section.
* **Set Custom Color To Show Product Discount On Shop Page**
Gives admin the freedom to set color in which customized text for product discount appear on shop page.
* **Set Custom Color To Show Product Discount On Product Page**
Gives admin the freedom to set color in which customized text for product discount appear on product page.
* **Set Custom Color To Show Product Discount On Sale Flash Section**
Gives admin the freedom to set color in which customized text for product discount appear on sale flash section.
* **Advanced Setting To Support Various Themes And Resolve Conflicts**
There is a setting available to render custom css to support various theme if any issue arises. This is advised to contact us to use this setting in best way possible, in case you face any issue.
* **Highly Customizable Settings**
So much settings are available to customize the usage of plugin, like custom text to show product discount, set your own color for text, etc.
* **Highly Intuitive Plugin**
Plugin is so much user-friendly that you must fall in love with it.
 
== Installation ==

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of "Woo Show Product Discount", log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Woo Show Product Discount" and click Search Plugins. Once you've found our "Woo Show Product Discount plugin" you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =
The manual installation method involves downloading our "Woo Show Product Discount" plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
